Common Nginx Use Cases
Reverse Proxy

A reverse proxy is a server that sits between client devices and a web server, forwarding client requests to the appropriate server and returning the server's responses to the clients. Use the Proxy Server template.

Load Balancer

Nginx can distribute incoming traffic across multiple servers to balance the load. This improves performance and reliability. Use the Advanced template.

SSL/TLS Termination

Nginx can handle SSL/TLS encryption and decryption, reducing the burden on your application servers. Create a custom configuration with the Advanced template.

Static Content Serving

Nginx excels at efficiently serving static files like HTML, CSS, JavaScript, and images. Use the Static Site template.

Automatic Configuration Process
Streamlined Configuration Process

When creating a new site configuration, the system will automatically:

  1. Create and save the configuration file
  2. Test the configuration for errors
  3. Enable the configuration if the test passes
  4. Reload Nginx to apply the changes immediately
  5. Redirect you to the appropriate management page:
    • For static sites: File Manager to upload and manage your files
    • For proxy and advanced configurations: Configuration details page

This automated workflow ensures your sites are properly configured and immediately available after creation.
